help.pathView

  • F2 - edit folder name
  • Shift + F6 - edit folder name
  • Return - rename folder (when in editing mode)
  • Esc - do not rename folder (when in editing mode)

help.imageViewer

  • Right Arrow - next picture
  • Left Arrow - previous picture
  • Esc - close image viewer
  • Cmd + F - full screen
  • Ctrl + F - full screen

help.imageViewerFullScreen

  • Right Arrow - next picture
  • Left Arrow - previous picture
  • Esc - close full screen
